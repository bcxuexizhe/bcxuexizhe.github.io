如何将时间戳转化为年月日时分秒格式

  1. 首先获取到当前的时间戳或者需要转化为时间的时间戳

    var time = new Date(时间戳);
//得到Thu May 11 2023 15:22:41 GMT+0800 (中国标准时间)
//这种样式的时间但是不是我们想要的所以要继续处理

  2. 然后使用getFullYear、getMonth、 getDate、getHours、getMinutes、getSeconds等方法来获取当前时间的年月日时分秒

    var y=time.getFullYear();//返回年份
//getMonth方法从 Date 对象返回月份 (0 ~ 11)，返回结果需要手动加一var d = time.getDate();
// getDate方法从 Date 对象返回一个月中的某一天 (1 ~ 31)
var M=time.getMonth()+1;
var d=time.getDate();
var h=time.getHours();
var m = time.getMinutes();

  3. 最后使用字符串拼接的方式得到我们想要的时间

    var times = y + '-' + M + '-' + d + ' ' + h + ':' + m + ':' + s
//得到这种格式2023-5-11 15:32:29

  4. 如果往后端传时间有严格要求必须是0000-00-00 00:00:00这种格式再做处理

    if (M <= 9) {
	M = '0' + M
}
if (d <= 9) {
	d = '0' + d
}
if (h <= 9) {
	h = '0' + h
}
if (m <= 9) {
	m = '0' + m
}
if (s <= 9) {
	s = '0' + s
}
//得到这种格式2023-05-11 15:35:35
