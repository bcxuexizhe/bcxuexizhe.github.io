Git 学习笔记

像 git commit 分支名字 或 git merge 分支名字这个命令后面最好加上 -m "消息"，如果你不加上 -m "消息"的话，它会打开一个程序让你去加上消息，这个程序还是在控制台里面，只不过是 Linux 里面一个叫做 VIM 的程序，它类似于 Windows 里面的记事本软件。

VIM 的操作：

按 i 进入插入模式，比如可以写入“修改了 a.txt”。
按 Esc 退出插入模式。
按 Shift+ZZ 退出 VIM 编辑器。
