位1的个数-力扣

题目

答案

class Solution {
    public int hammingWeight(int n) {
        int a = 0;
        for(int i =0;i<32;i++){
            if((n &  (1<<i))!=0){
                a++;
            }
        }
        return a;
    }
}

解析

遍历32位二进制位，然后和1相与看其是否是1，统计个数即可
