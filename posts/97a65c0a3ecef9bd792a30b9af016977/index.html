eclipse创建第一个java web项目并运行

为了能编写java web项目，建议安装支持javaee开发的eclipse版本。

1.下载eclipse

地址：

https://www.eclipse.org/downloads/packages/release/2021-03/r

2.解压后启动eclipse

3.新建java web工程

设置项目名称，指定tomcat的版本及路径

4. 添加一个jsp页面

在jsp中如输入如下代码：

<%

String str = "hello world!";

out.println(str);

%>

5. 运行项目

修改配置

运行之前需要先修改一个设置，否则访问会有问题

运行

在项目上点击右键 run as > Run on server

如果你愿意，可以在浏览器访问 你的web项目
